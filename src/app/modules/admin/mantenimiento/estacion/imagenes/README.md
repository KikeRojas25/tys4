## 📝 Instrucciones para Agregar Imágenes

1. Toma las capturas de pantalla según la lista anterior
2. Guarda las imágenes en esta carpeta con los nombres indicados
3. Formato recomendado: PNG o JPG
4. Tamaño recomendado: Ancho máximo de 1200px para mejor visualización
5. El manual se actualizará automáticamente con las referencias a estas imágenes

## 🎨 Consejos para las Capturas

- Usa un navegador con la interfaz limpia (sin extensiones visibles)
- Asegúrate de que los datos de ejemplo sean claros y legibles
- Evita información sensible o datos reales de clientes
- Usa datos de prueba o genéricos
- Captura la pantalla completa o la sección relevante
